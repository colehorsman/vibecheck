# Beginner Requirements

What you need to start vibe coding as a complete beginner.

## Essential (Required)

- [ ] **A browser** - Chrome, Firefox, Safari, or Edge
- [ ] **An email address** - For creating accounts
- [ ] **A project idea** - Or use one of our starters

## Recommended (Optional)

- [ ] **GitHub account** - Free, useful for saving code later
- [ ] **Google account** - Required for some tools
- [ ] **30-60 minutes** - For your first project

## Not Required

- ❌ Coding knowledge
- ❌ Software installation
- ❌ Command line experience
- ❌ Money (free tiers are enough)

---

## Tool Accounts to Create

Create these accounts before your first session:

| Tool | Sign Up | Why |
|------|---------|-----|
| **Lovable** | [lovable.dev](https://lovable.dev) | Main building tool |
| **v0** | [v0.dev](https://v0.dev) | UI components |
| **GitHub** | [github.com](https://github.com) | Save your code |
| **Vercel** | [vercel.com](https://vercel.com) | Deploy for free |

**Time to set up:** ~10 minutes

## Project Ideas (Pick One)

### Easy (15-20 min)
- Personal portfolio page
- Simple landing page
- Link-in-bio page (like Linktree)

### Medium (30 min)
- Todo app
- Recipe organizer
- Habit tracker
- Reading list manager

### Stretch (45-60 min)
- Simple dashboard
- Contact form with email
- Quiz game

- [ ] Browser open
