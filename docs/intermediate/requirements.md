## Setup Checklist

### 1. Install Tools

```bash
# Check Node version (need 18+)
node --version

# Install pnpm (faster than npm)
npm install -g pnpm

# Install Cursor
# Download from cursor.com
```

### 2. Create Accounts

| Service | Link | Purpose |
|---------|------|---------|
| GitHub | [github.com](https://github.com) | Version control |
| Supabase | [supabase.com](https://supabase.com) | Database + Auth |
| Vercel | [vercel.com](https://vercel.com) | Deployment |
| Cursor | [cursor.com](https://cursor.com) | AI IDE |

### 3. Configure Git

```bash
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
```

### 4. Set Up SSH Key (for GitHub)

```bash
ssh-keygen -t ed25519 -C "your@email.com"
# Add to GitHub: Settings > SSH Keys
```
