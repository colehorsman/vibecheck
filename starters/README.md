# Starter Projects

Quick-start templates for common project types.

## Available Starters

| Starter | Stack | Time | Level |
|---------|-------|------|-------|
| [nextjs-supabase](nextjs-supabase/) | Next.js 14 + Supabase + Tailwind | 5 min | Intermediate |
| [python-fastapi](python-fastapi/) | FastAPI + SQLite + Pydantic | 5 min | Intermediate |
| [chrome-extension](chrome-extension/) | Vanilla JS + Chrome APIs | 10 min | Beginner |

## How to Use

### Option 1: Copy Files

1. Navigate to the starter folder
2. Copy files to your project
3. Follow the README in each starter

### Option 2: Use as Reference

1. Read the structure
2. Ask AI to create similar setup
3. Customize for your needs

## What's Included

Each starter includes:
- ✅ Project structure
- ✅ Configuration files
- ✅ Environment setup (.env.example)
- ✅ Basic functionality
- ✅ README with next steps
