## How to Use

1. Copy the prompt template
2. Fill in the `[PLACEHOLDERS]`
3. Paste into your AI tool
4. Iterate as needed

## Tips

- Be specific about what you want
- Include relevant context
- Specify constraints upfront
- Ask for explanations when learning
